# mediawikirag

A Python project managed with [uv](https://github.com/astral-sh/uv).

## Setup

Install uv:
```
pip install uv
```

Install dependencies:
```
uv pip install -r requirements.txt
```

## Project Structure
- `src/` — Source code
- `tests/` — Test code

## Usage
Add your main code in `src/main.py` and tests in `tests/test_main.py`.
